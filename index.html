MP4-PNG

MP4 위에 PNG 아이콘을 올리고, 드래그/정렬 후 영상으로 저장하세요.

Alignment

출력해상도

업스케일은 출력 픽셀 수를 키우지만, 원본 영상보다 실제 디테일이 늘어나지는 않습니다.

레이어

PNG 아이콘을 업로드하면 여기서 크기를 조절할 수 있습니다.

동영상과 아이콘을 준비하면 내보내기를 시작할 수 있습니다.

작업 영역

MP4를 선택하면 원본 크기로 표시됩니다.

Excel to JSON

엑셀 파일을 올리면 시트별 JSON으로 변환해서 바로 다운로드할 수 있습니다.

파일을 업로드하면 JSON 미리보기와 다운로드 버튼이 활성화됩니다.

JSON 미리보기

첫 번째 시트의 변환 결과 일부가 여기에 표시됩니다. 

아직 업로드된 엑셀 파일이 없습니다.

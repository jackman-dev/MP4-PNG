# MP4 아이콘 정렬 에디터

MP4 동영상 위에 PNG 아이콘을 여러 개 올리고, 드래그 배치와 자동 정렬 후 결과 영상을 다운로드할 수 있는 브라우저 기반 도구입니다.

## 실행 방법

정적 파일 앱이라서 간단한 로컬 서버로 실행하면 됩니다.

```bash
python3 -m http.server 8000
```

그 다음 브라우저에서 아래 주소를 여세요.

```text
http://localhost:8000
```

## 지원 기능

- MP4 동영상 업로드
- PNG 아이콘 여러 개 업로드
- 업로드 이후 추가 업로드 반복 가능
- 아이콘 드래그 앤 드롭 배치
- `좌측 정렬 (30px)` 기능
- `수직 분배` 기능
- 레이어별 개별 크기 조절
- 원본 영상 크기 그대로 작업 영역 표시
- 큰 영상은 스크롤로 편집 가능
- 브라우저 지원 형식으로 결과 영상 다운로드

## 참고

- 브라우저가 MP4 녹화를 지원하면 `.mp4`로 저장됩니다.
- MP4 녹화를 지원하지 않는 브라우저에서는 `.webm`으로 저장될 수 있습니다.
